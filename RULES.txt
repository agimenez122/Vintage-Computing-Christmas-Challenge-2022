#Formato

Envíe lo siguiente dentro de un archivo zip :

una imagen (captura de pantalla)  del resultado (del objeto) [p. ej. result.png]
Archivo ejecutable (por ejemplo, código ensamblador compilado, archivo BASIC que se ejecuta en el host, etc.) [por ejemplo, star.prg, star.bin, star.com, ...]

Código fuente:
código fuente como archivo de texto legible (por ejemplo, código BASIC, código ensamblador, etc.) [por ejemplo, source.txt]
una imagen del código fuente enumerado (en la máquina nativa o en la máquina codificada); se puede integrar con la imagen del resultado (opcional) [ej. source.png]
